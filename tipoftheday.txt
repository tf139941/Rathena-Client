→/savechat  :  把對話視窗的內容儲存起來
#
→按住ctrl鍵，再按*鍵，可把微型地圖從半透明變成不透明，或是不透明變成半透明 
#
→按住shift鍵，再按滑鼠右鍵拖曳，可以把視角的角度抬高，如要恢復原狀，按住shift鍵，再按兩次滑鼠右鍵，既可恢復 
#
→按住ctrl鍵，再按滑鼠右鍵拖曳，可以把視角放大縮小，這時按住ctrl鍵，再按兩次滑鼠右鍵，既可恢復原狀，用滑鼠滾輪也可以把畫面放大縮小
#
→把游標放在敵人身上時，游標會變成刀， 既可攻擊
#
→從怪物裡得到的經驗值，是與怪物受到的損害值同正比， ※一起給攻擊怪物時，先攻擊怪物的人，得到的經驗值較多
#
→按住ctrl鍵攻擊怪物時，會變成[鎖定攻擊狀態]，將會持續自動攻擊到魔物死亡為止，想要解除自動攻擊狀態時，將滑鼠游標點選其他位置既可
#
→想要撿掉下來的物品，把游標指在物品上再[點選滑鼠左鍵]即可，撿到的物品會放在物品欄裡
#
→把游標移到物品身上[點選滑鼠右鍵]時，可以看到物品的情報
#
→玩家腳底下二個橫桿是玩家的HP值和SP值，綠色的表示HP值，藍色的表示SP值，HP值到0時，表示玩家已死亡，為了警惕玩家HP值剩下25%以下時，HP值會改變成顯示為紅色
#
→EXP顯示桿已滿的話(100%)，基礎等級會上升，基礎等級上升時，會獲得素質點數（Status Point），玩家可自由加給於想培養的素質
#
→EXP 表示桿下面的JOBEXP顯示桿已滿的話(100%)， 職業等級會上升，會獲得技能點數，玩家可自由加給於想培養的技能
#
→若玩家不動於原地時，HP和SP會稍微恢復※坐著時會恢復速度會加快兩倍，而當重量超過50%以上時，將不會恢復HP值與SP值
#
→Weight(重量) : 顯示為最大負重量和負重量，玩家無法攜帶超過最大負重量的物品，物品重量超過最大負重量時，將不能攜帶任何物品。若攜帶的物品重量超過最大負重量一半時限度超過一半時，將會顯示為紅色，這時HP和SP不會自動恢復，還有超過最大負重量90%時，玩家將不能攻擊和使用技能
#
→Zeny(金幣) : 金幣是在仙境傳說裡所使用的[金錢]單位 ※玩家所擁有的金幣當交易或購買物品時會自動扣減
#
→基礎等級上升後，會得到素質點數（Status Point）， 玩家可利用這點數培養素質
#
→STR(力量) : 對玩家攻擊力有直接的影響 ，力量增加時，最大負重量也將會增加，將可攜帶更多物品。力量增加將會增加攻擊力
#
→DEX(靈巧) : 對玩家使用武器時有直接的關係 ，數值越高，越能發揮武器的攻擊力，用弓箭攻擊時，因DEX值提高也將會提高攻擊力，也會增加命中率，也影響到施放魔法的速度
#
→AGI(敏捷) : 對玩家的行動敏捷度有直接的關係，增加這數值時，迴避率將會增加，可閃躲敵人的攻擊。而各種武器類的攻擊速度也將會提升
#
→VIT(體質) : 對玩家的防禦力和HP值多寡有直接的關係 ，數值越高，防禦力也將會增加，遭受敵人攻擊時損失可減低，還有將會增加最大HP值和加快恢復HP的速度
#
→INT(智力) : 對玩家的魔力（SP）有直接的關係，數值越高，將會增加魔法攻擊力（MATK）和魔法防禦力（MDEF）
#
→LUK(幸運) : 玩家的幸運對各種狀況都有直接的關係，主要是必殺攻擊率和完全迴避率，都和幸運增加有直接關係
#
→[Print Screen] or [Scroll Lock] : 拍照，照片會在[ScreenShot] 檔案內存檔起來
#
→[F10] : 可以調查對話視窗大小
#
→[F11] : 只保留基本情報視窗和對話視窗，關閉其他視窗
#
→[F12] : 可以開啟關閉快速使用列  ※將經常使用的物品或是技能拖曳於在F1∼F9鍵上，直接按所屬快速鍵就可以使用
#
→[INSERT] :將可坐下或站立  ※將[/坐下]或是[/站立]這兩個指令直接輸入於對話窗就可以使用。在坐下的狀態時HP、SP回復速度比站立時加快兩倍
#
→[Alt] + [End] : 可以把角色下方 [HP / SP] 圖示開啟/關閉
#
→[Alt] + [Home] : 可以把可移動的地區圖示開啟/關閉
#
→[Alt] + [=] : 切換輸入狀態，全形轉半形。用全形輸入的命令不會執行 這時候切換成半形輸入即可
#
→[Alt] + [F10] : 可以把聊天視窗開啟/關閉
#
→[Alt] + [1~9] , [Ctrl] + [-,=,w] :[Alt] + [1~9] 可以使用出以下表情符號︰驚嘆號、問號、音符、愛心、汗滴、靈感燈泡、怒氣青筋、烏雲、金錢、無言點點。[Ctrl] + [-,=,w]︰ 剪刀、石頭、布 
#
→/WHO : 可以看到伺服器上， 線上玩家總人數
#
→/WHERE : 可以看到現在所在位置的情報和座標
#
→/H : 顯示對基本的指令說明
#
→/BGM : 可以把背景音樂開啟/關閉  ※也可以按 [Alt] + [0] 在視窗上執行
#
→/SOUND : 可以把效果音開啟/關閉   ※也可以按 [Alt] + [0] 在視窗上執行
#
→/EFFECT : 可以把遊戲裡的各種效果圖示開啟/關閉   ※關閉效果時，可以使遊戲執行上更加順暢
#
→/SKIP :開啟/關閉畫面同步更新
#
→/EX  【角色名稱】:  可以隔絕此玩家所說的話和悄悄話
→/IN 【角色名稱】:  恢復與此玩家之間的對話
#
→/EXALL : 把所有角色的對話隔絕
→/INALL : 恢復與所有玩家對話
#
→/EX : 顯示被隔絕玩家的清單
#
→/ORGANIZE 【隊伍名稱】: 可組成隊伍  ※如是隊長時，可以邀請玩家參加組隊
#
→把游標放在想邀請的玩家身上， 點選滑鼠右鍵，會顯示 ' 邀請參加組隊 ' 選項，利用這選項，可組成隊伍  
#
→/LEAVE : 可以脫離自己參加的隊伍  ※打了此項指令後，將不會詢問同意與否
#
→/EXPEL 【角色名稱】: 隊長可將隊員於隊伍中剔除※打了此項指令後，將不會詢問同意與否
#
→點選對話視窗右下段 [圓圈]後，將會顯示〝隊伍發言頻道〞，點選後將可使用隊伍頻道與隊員對談
#
→%  : 在發言前加上%既可直接使用隊伍頻道與隊員對談，或是按住Ctrl鍵，再按Enter鍵，把輸入的內容直接傳達給隊員  
#
→隊員右上方微型地圖將會顯示各隊員所在位置
#
→悄悄話 : 聊天視窗左邊輸入角色名稱，右邊輸入對話內容 ，用TAB鍵將可快速於二個輸入欄裡轉換位置
#
→選擇微型地圖 [+ / -] 符號時，地圖將會 [放大 / 縮小] 
#
→[MVP系統] : 與Boss級魔物對決時，為了保護最高傷害者的權益，魔物的MVP物品將會直接掉落於MVP玩家，如魔物未掉落MVP物品，將補償玩家50%的經驗值 
#
→遊戲在視窗模式進行時，可以移動視窗位置，按住[Alt] + [空白鍵]再按[m]鍵，可用方向鍵或是滑鼠調整位置
#
→/fog : 可以把霧化效果開啟/關閉，霧化效果是為各種地圖做天氣表現使用，對低階電腦配備玩家來說，霧化效果會增加電腦負擔，玩家如被迫跳出或停止遊戲時，這時建議玩家關掉霧化效果，霧化效果也可組態設定下永久開啟或是永久關閉
#
→從魔物獲得的物品、不能確認物品的效果，可用道具商店所販賣的放大鏡來確認物品的效果
#
→獵人若要用陷阱系列技能，身上必須《攜帶裝置用陷阱物品》
#
→組成工會需要消耗一個華麗金屬，組成工會指令為  /GUILD  [工會名字] ,工會名字不可有特殊字元與空白用，Alt +G 可以打開工會視窗
#
→工會會員之間對談可使用/GC 【對話內容】 或是 $ 【對話內容】 或是按 Alt + Enter 也可以
#
→若想使裝備附屬卡片，請先卸下所想附屬的裝備，對卡片點選兩下將會出現可附屬物品清單，再點選所想附屬之物品既可
#
→有關禮貌點數系統說明   
--初學者技能在基礎技能等級8時可使用
--對玩家角色，用按下滑鼠右鍵，會顯示給予禮貌點數 
#
→對 幾種異常狀態效果說明
--昏眩 : 變成昏眩狀態，將不能行動
--黑暗 : 畫面變黑暗，對怪物來說視線會變狹礙
--冰凍 : 將無法行動，當受到攻擊時，才會解除此狀態
#
→死亡時經驗值會下降，使用技能後或是戰鬥狀態後，需等待5秒才可下線
#
→使用倉庫，儲存紀錄點，租用手推車，詢問卡普拉職員即可
